# DynoAI Landing Page

The landing page for DynoAI - AI-powered dyno software for Harley-Davidson EFI tuning.

## Features

- Modern, responsive design
- Clean layout with hero section, features, articles, and contact form
- Professional styling with blue/purple gradient accents
- Fixed navigation with smooth scrolling

## Files

- `dynoai-landing.html` - Main landing page
- `styles.css` - Main stylesheet for the landing page
- `article-styles.css` - Styles for article pages (if created)

## Usage

Simply open `dynoai-landing.html` in a web browser to view the page locally.

For production deployment, upload all files to your web server.

## Technology

- HTML5
- CSS3 with custom properties
- Google Fonts (Inter)
- Responsive design with mobile breakpoints
