## Overview

DynoAI is revolutionizing EFI tuning with intelligent VE correction and real-time knock detection. This repository contains the marketing landing page showcasing DynoAI's capabilities.

### Key Features Highlighted

- **One-Pull Baseline™** — Generate accurate VE tables from a single WOT pull
- **Real-Time Knock Detection** — Multi-modal analysis for early problem detection
- **Intelligent Guidance** — AI-recommended test points and optimization paths
- **Deep Diagnostics** — Distinguish between fueling, timing, and mechanical issues
- **Automatic Reporting** — Professional PDF reports generated automatically
- **Institutional Memory** — Build knowledge from every tune

## Quick Start

### View Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/dynoai-landing.git
   cd dynoai-landing
   ```

2. Open `index.html` in your browser:
   ```bash
   # macOS
   open index.html
   
   # Windows
   start index.html
   
   # Linux
   xdg-open index.html
   ```

### Development

Simply edit the HTML/CSS files and refresh your browser to see changes.
